# TerrariaBot
基于 mirai 的 Terraria TShock QQ 机器人   

***

## 声明

**本项目仅用于学习，请勿用于非法用途**

***

<details>

+ 远程指令

</details>

# 开始使用
+ ## 安装[Mcl](https://github.com/iTXTech/mirai-console-loader)

+ 解压打开

+ 启动mcl.cmd

+ 将下载好的 TerrariaRobot-1.0.jar 放进 plugins 内

+ 成功运行后自行配置登录信息等

+ 插件配置文件自动生成于目录 data\com.example.TerrariaRobot
